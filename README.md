## 🧩 它是什么

校园AI站是一个 **AI Skills 应用商店**——

> 就像 App Store 让你下载手机应用，校园AI站让你**浏览、搜索、一键安装** AI 编程技能。

每个"技能"就是一个 `SKILL.md` 文件，里面包含特定领域的工作流和知识。安装到你的 AI 编程工具后，AI 就能在该领域表现得更专业。

### 数据流示例：用户点赞一个 Skill

```
1. 用户点击 ❤️ 
2. → React 组件 handleLike()
3. → Supabase JS Client (HTTPS)
4. → UPDATE skills SET likes = likes + 1 WHERE slug = ?
5. → 实时更新 UI（optimistic update）
6. → localStorage 记录（防止重复点赞）
```

## 🚀 快速开始

### 前置要求

- **Node.js** >= 18.x
- **pnpm** >= 8.x

### 本地开发

```bash
# 1. 克隆
git clone https://github.com/JIAChenyu525/ai-skills-hub-v2.git
cd ai-skills-hub-v2/front_design

# 2. 安装
pnpm install

# 3. 启动
pnpm dev
# → http://localhost:5173
```

### 构建

```bash
pnpm build
# 输出 → dist/
```

### 数据库

1. 注册 [Supabase](https://supabase.com)
2. 创建项目 → 获取 Project ID 和 Anon Key
3. 在 Supabase SQL Editor 执行建表语句（[见数据库设计](#-数据库设计)）
4. 修改 `utils/supabase/info.tsx` 填入你的凭证

## 🗄️ 数据库设计

<details open>
<summary><b>skills 表</b> — 核心技能数据</summary>

| 列 | 类型 | 说明 |
|------|------|------|
| `id` | UUID PK | 唯一标识 |
| `slug` | TEXT UNIQUE | URL 标识，如 `paper-writer` |
| `name` | TEXT | 技能名称 |
| `description` | TEXT | 简短描述 |
| `full_content` | TEXT | SKILL.md 完整内容 |
| `category` | TEXT | 论文写作/求职就业/编程开发/考试备考/其他 |
| `author_name` | TEXT | 作者名 |
| `author_github` | TEXT | GitHub 用户名 |
| `install_command` | TEXT | 安装命令 |
| `repo_url` | TEXT | GitHub 仓库链接 |
| `version` | TEXT | 版本号 |
| `status` | TEXT | pending / approved / rejected |
| `likes` | INTEGER | 点赞数 |
| `downloads` | INTEGER | 安装下载数 |
| `created_at` | TIMESTAMPTZ | 创建时间 |
| `updated_at` | TIMESTAMPTZ | 更新时间 |

</details>

<details>
<summary><b>其余 5 张表</b> — profiles / comments / ratings / reports / notifications</summary>

| 表 | 关键列 | 约束 |
|------|------|------|
| `profiles` | user_id (FK→auth.users), username, avatar_url, bio | user_id UNIQUE |
| `comments` | skill_slug, user_id, content, created_at | 按 skill_slug + created_at 索引 |
| `ratings` | skill_slug, user_id, rating (1-5) | UNIQUE(skill_slug, user_id) |
| `reports` | skill_slug, user_id, reason, created_at | — |
| `notifications` | user_id, type, title, body, skill_slug, read | 按 user_id + created_at 索引 |

全部启用 **Row Level Security**，默认策略 `FOR ALL USING (true)`。

</details>

## 📦 部署

pnpm build

<details>
<summary><b>Q: 为什么有时候打不开？</b></summary>

网站托管在 GitHub Pages，国内大部分地区可访问。如果遇到问题：换个浏览器、开无痕模式、或用手机移动数据试试。
</details>

<details>
<summary><b>Q: 需要 Claude Code 才能用吗？</b></summary>

不需要！支持 Trae、Cursor、Codex、Windsurf、Copilot 等。教程页有各工具配置指南。
</details>

<details>
<summary><b>Q: 搬运别人的 Skill 合法吗？</b></summary>

只收录开源协议（MIT/Apache/GPL）的 Skill，标注原作者和来源链接。侵权请用举报功能。
</details>

<details>
<summary><b>Q: 数据是真实的吗？</b></summary>

所有点赞、下载、评分数据来自真实用户行为。没有任何虚构数据。
</details>

<details>
<summary><b>Q: 个人资料怎么设置？</b></summary>

登录 → 点右上角头像 → 个人设置 → 改头像/用户名/简介 → 保存。
</details>
